# ch58x-SDK

This repository synchronizes with [openwch/ch583](https://github.com/openwch/ch583), extracting and reorganizing the SDK components for easier development.
> **Although this repository is named "ch58x-SDK", it is NOT compatible with the CH585 series, history related issue.**  
> If you need the SDK for CH585, please refer to the dedicated repository:  
> [openwch/ch585](https://github.com/openwch/ch585)

All source files have been converted to UTF-8 encoding for improved cross-platform compatibility.
